Projet Game and Watch

Ce jeu très simple est inspiré des jeux WarioWare et de leurs mécaniques principales.
Le but du jeux est simple, réussir différents minijeux (pour l'instant 2) afin de faire monter notre score. Chaque mini-jeu réussit augmente notre score de 1 point, mais si on ne le réussit pas dans le temps imparti ou qu'on le rate alors on perd une vie.
Le joueur possède 3 vies, une fois arrivé à 0, c'est le game over. Un éran apparait alors en indiquant le score final, qui sera ensuite renseigné dans un scoreboard accessible dans le menu principal.
Ce scoreboard indique les 5 meilleurs scores atteints durant toutes les parties faites (on ne peut pas choisir de nom à assigner à chaque score pour l'instant)
Le jeu est pour le moment composé de 2 minijeux simples :
- 1 minijeu où on doit appuyer au bon moment (quand une barre blanche se situe dans une zone verte)
- 1 minijeu simple de type GameAndWatch où le but est de déplacer un personnage de sorte à ce qu'il récupère un objet qui descend petit à petit

Une fois ces 2 minijeux réalisés, la difficulté augmente d'un niveau. La difficulté à un niveau max qui est 3 (1 premier niveau de difficulté de base, un niveau 2 quand on réussit les 2 minijeux, puis un niveau 3 quand on réussit encore une fois les 2 minijeux). Une fois ce niveau de difficulté 3 atteint, le joueur peut continuer les minijeux à l'infini tant qu'il a encore des vies afin de faire le meilleur score possible.

Le joueur sait quand il échoue le minijeu car un sound effect négatif se fait entendre et qu'il voit son nombre de vies baisser.
Il sait également quand il réussit le mini jeu car un sound effect positif se fait entendre et qu'il voit son score augmenter.
